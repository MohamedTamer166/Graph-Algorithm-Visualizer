Kruskal's Algorithm Solver


Usage

  • Use Draw vertex then click on the canvas to add vertices.
  • Use Draw edge, then click two vertices to connect them with an undirected edge.
  • Use Set cost or label to edit edge weights (must be positive) or vertex labels.
  • Click Calculate MST (Kruskal's) to sort all edges, select valid edges, and highlight the MST in orange.
  • The result panel shows sorted edges and whether each was selected or skipped.
  • If the graph is disconnected, the solver shows Graph is disconnected - MST not possible.
